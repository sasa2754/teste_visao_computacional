## Bibliotecas necessárias:
- Numpy
- Matplotlib
- OpenCV

## Comandos para instalação:
```
pip install numpy
pip install matplotlib
pip install opencv-python
```

## Importes no código:
```
import numpy
import matplotlib
import cv2 as cv
```


## Configurando a raspberry Pi:
```
sudo apt update
sudo apt upgrade
sudo apt install python3-opencv
```
